rand : term -> term

SYNOPSIS
Returns the operand from a combination (function application).

DESCRIPTION
rand `t1 t2` returns `t2`.

FAILURE CONDITIONS
Fails with rand if term is not a combination.

EXAMPLE
  # rand `SUC 0`;;
  val it : term = `0`
  # rand `x + y`;;
  val it : term = `y`
