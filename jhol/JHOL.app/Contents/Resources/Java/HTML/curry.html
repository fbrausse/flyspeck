curry : ('a * 'b -> 'c) -> 'a -> 'b -> 'c

SYNOPSIS
Converts a function on a pair to a corresponding curried function.

DESCRIPTION
The application curry f returns \x y. f(x,y), so that 
   curry f x y = f(x,y)

FAILURE CONDITIONS
Never fails.

EXAMPLE
  # curry mk_var;;
  val it : string -> hol_type -> term = 
  # it "x";;
  val it : hol_type -> term = 
  # it `:bool`;;
  val it : term = `x`
