is_var : term -> bool

SYNOPSIS
Tests a term to see if it is a variable.

DESCRIPTION
is_var `var:ty` returns true. If the term is not a variable the result is false.

FAILURE CONDITIONS
Never fails.

EXAMPLE
  # is_var `x:bool`;;
  val it : bool = true
  # is_var `T`;;
  val it : bool = false
