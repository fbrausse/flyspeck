itlist : ('a -> 'b -> 'b) -> 'a list -> 'b -> 'b

SYNOPSIS
List iteration function. Applies a binary function between adjacent elements of a list.

DESCRIPTION
itlist f [x1;...;xn] y returns 
   f x1 (f x2 ... (f xn y)...)
It returns y if list is empty.

FAILURE CONDITIONS
Never fails.

EXAMPLE
  # itlist (+) [1;2;3;4;5] 0;;
  val it : int = 15
  # itlist (+) [1;2;3;4;5] 6;;
  val it : int = 21
