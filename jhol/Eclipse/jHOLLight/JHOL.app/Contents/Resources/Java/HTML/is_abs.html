is_abs : term -> bool

SYNOPSIS
Tests a term to see if it is an abstraction.

DESCRIPTION
is_abs `\var. t` returns true. If the term is not an abstraction the result is false.

FAILURE CONDITIONS
Never fails.

EXAMPLE
  # is_abs `\x. x + 1`;;
  val it : bool = true

  # is_abs `!x. x >= 0`;;
  val it : bool = false
