mk_imp : term * term -> term

SYNOPSIS
Constructs an implication.

DESCRIPTION
mk_imp(`t1`,`t2`) returns `t1 ==> t2`.

FAILURE CONDITIONS
Fails with mk_imp if either t1 or t2 are not of type `:bool`.

EXAMPLE
  # mk_imp(`p /\ q`,`r:bool`);;
  val it : term = `p /\ q ==> r`
