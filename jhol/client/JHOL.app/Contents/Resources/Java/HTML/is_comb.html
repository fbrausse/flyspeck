is_comb : term -> bool

SYNOPSIS
Tests a term to see if it is a combination (function application).

DESCRIPTION
is_comb "t1 t2" returns true. If the term is not a combination the result is false.

FAILURE CONDITIONS
Never fails

EXAMPLE
  # is_comb `x + 1`;;
  val it : bool = true
  # is_comb `T`;;
  val it : bool = false
