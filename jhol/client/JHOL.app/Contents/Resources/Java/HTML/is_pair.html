is_pair : term -> bool

SYNOPSIS
Tests a term to see if it is a pair.

DESCRIPTION
is_pair `(t1,t2)` returns true. If the term is not a pair the result is false.

FAILURE CONDITIONS
Never fails.

EXAMPLE
  # is_pair `1,2,3`;;
  val it : bool = true

  # is_pair `[1;2;3]`;;
  val it : bool = false
