parses_as_binder : string -> bool

SYNOPSIS
Tests if a string has binder status in the parser.

DESCRIPTION
Certain identifiers c have binder status, meaning that `c x. y` is parsed as a shorthand for `(c) (\x. y)'. The call parses_as_binder "c" tests if c is one of the identifiers with binder status.

FAILURE CONDITIONS
Never fails.

EXAMPLE
  # parses_as_binder "!";;
  val it : bool = true
  # parses_as_binder "==>";;
  val it : bool = false
